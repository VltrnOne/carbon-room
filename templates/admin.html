CLI Access Instructions

Team members can invoke protocols directly from the terminal without seeing the content.

curl -X POST http://localhost:8003/api/invoke \ -H "Content-Type: application/json" \ -d '{"keyword": "PROTOCOL_NAME", "user_id": "your_name"}'

Or use the quick start script:

python quick_start.py list python quick_start.py invoke GOD_MODE_WIZARD your_name
